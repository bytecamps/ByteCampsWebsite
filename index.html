About Us

Hi! We are high schoolers at Pinewood in Los Altos and we enjoy programming. We would love to teach this to kids (ages 8-15)! We think that programming should be fun and strive to teach it that way. This is not aimed to be a professional camp but rather a fun, small camp tutoring service.

NOTE: All camps will be offered online via Zoom.

Receive a FREE wireless charger with every camp! Sign up now!

iOS Development Camp

Target audience: kids 9-15 years old

During this camp you will learn how to make your very own iOS apps using Apple's Swift and Xcode. No programming experience is required.

You will learn what Swift and Xcode are. You will also learn about programming fundamentals like variables, loops, if statements, functions, lists, etc. Throughout the week, multiple apps will be built to show off! For this camp, we are using Apple's new SwiftUI.

Requirements:

  • A macbook w/ Xcode installed (instructions follow later)

Pricing: $199 a week/term

Advanced iOS Development Camp

Target audience: everyone who has completed the iOS App Development Camp

During this camp you will learn how to make even more advanced apps using Swift and Xcode.

We will dive deeper into Swift and make a fun color game with leaderboard. For the leaderboard, we will even use the Firebase Database. We will also learn how to make more complex screens using SwiftUI.

Requirements:

  • A macbook w/ Xcode installed

Pricing: $199 a week/term

Web Development Camp

Target audience: kids 8-15 years old

During this camp you will learn how to make your own websites using HTML, CSS, and Javascript. No programming experience is required!

You will learn what HTML, CSS, and Javascript are and how to use them! You will also learn programming fundamentals and style your websites to make them look amazing. We will even dive into the framework Bootstrap! Multiple websites will be made throughout the week!

Requirements:

  • A computer w/ Atom installed (instructions follow later)

Pricing: $199 a week/term

Procreate Camp

Target audience: kids 8-13 years old

Do you want to know how to use the app Procreate to draw? During this camp you will learn the ins and outs of Procreate. You do not have to be a star at art to make fun projects with this app.

You will learn how to navigate Procreate. You will also learn about different brushes, layers, symmetry, liquidfication (marbling), etc. It will be combined with fun projects!

Requirements:

  • An ipad
  • the app Procreate (around ten dollars)
  • Optional: Apple Pencil

Pricing: $199 a week/term

Junior Coding Camp w/ Scratch and Python

Target audience: kids 7-10 years old

Do you want to get started with Programming? During this camp you will be introduced to programming using Scratch and Python.

We will use Scratch for the first half of the week and make some really nice (visual) projects! Then, we will move to Python and write our first lines of code!

Requirements:

  • A computer
  • Some experience with Scratch is preferred

Pricing: $199 a week/term

Games w/ Python Camp

Target audience: kids 8-15 years old

During this camp you will learn what Python is and how to program with it! No programming experience is required.

You will learn about programming concepts like variables, loops, if statements, functions, lists, etc. We will dive into turtle and have a competition on who can make the best art! We'll also make some games that you can play with Python! Throughout the week, we will do projects to keep everybody engaged. We end with a final project that you can show everybody!

Requirements:

  • A computer

Pricing: $199 a week/term

Frequently Asked Questions

Do you also offer tutoring?

Yes! We also offer tutoring. Send us an email or go to the sign up section and fill in the form!

Do I need a computer/iPad? If so, what kind?

What device you need depends on the camp. For the Python or Web Development camp, any computer is fine (both Windows & Apple). For the iOS App Development camp, you will need a macbook (no windows computers). Lastly, for the Procreate camp, you need an iPad with the app Procreate installed.

How many students are in a camp?

We have a maximum of three of four students per instructor.

Do I need any programming experience for a programming camp?

No, you can get started without any programming experience. The camp we would recommend to get started would be the Python camp, which teaches you everything you have to know to get started. The Web Development camp would be ideal after that (but can also be done with no/minimal experience). Lastly, the iOS App Development camp requires an understanding of programming.

Do I need any experience with Procreate for the Procreate camp?

Nope! You can take the camp with no experience, we will teach you all the techniques.
